### 基于PO模型设计思路 + airtest + pytest+poco实现的一个demo

按照基础操作层+页面层+数据层+测试用例层+报告的模式来编写的，

### 各文件层级

​	   base：	  文件夹封装是一些公用的方法，基本元素操作相关

 	  page：	  层是以每个页面为对象来封装的

​	   data：	   存放测试用例的数据

  	 case：	   放测试用例

​	   tools：	  存放公用类方法、读取json文件的函数、获取绝对路径的函数

​	report_log：存放报告和日志



### 基础环境: 

​	python 3.7.8，airtest 1.3.1



​	测试包名: com.kuke
